# 数据失败复盘模板

当一批数据导致训练失败或评估异常时，使用本模板。

## 1. 摘要

```text
事件名称：
发现日期：
影响模型：
影响数据版本：
严重级别：P0/P1/P2/P3
当前状态：已缓解 / 修复中 / 已回滚
```

## 2. 现象

```text
哪个指标异常？
什么时候开始？
影响哪些任务域？
是否影响线上？
```

## 3. 根因

```text
数据问题类型：
根因样本：
为什么质检没有发现：
是否和工具/环境/标注有关：
```

## 4. 影响范围

```text
受影响 dataset version：
受影响 checkpoint：
受影响 eval：
是否污染 regression set：
是否需要通知安全/合规：
```

## 5. 修复动作

```text
删除哪些数据：
降权哪些数据：
修哪些转换脚本：
新增哪些质检规则：
新增哪些 bad examples：
是否需要重训：
```

## 6. 复测

```text
复测数据版本：
必跑 eval：
修复前指标：
修复后指标：
是否达到回滚解除条件：
```

## 7. 防复发

```text
新增 schema 检查：
新增 rubric：
新增人工抽查项：
新增 regression case：
负责人：
截止日期：
```
